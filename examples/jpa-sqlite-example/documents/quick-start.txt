Quick Start Guide for Custom Repositories

1. Implement VortexCrudDataStore<FieldType, ModelClass>
   - This is your custom data access layer
   - Examples: FileSystemDataStore, MongoDataStore, RestApiDataStore

2. All methods you need to implement:
   - insertRecord() - Create or update
   - getRecordsFromTable() - List with pagination
   - getRecordById() - Find by ID
   - updateRecordById() - Update specific record
   - deleteRecordById() - Delete specific record
   - count() - Total count
   - Filtering methods for search

3. Register with VortexCrudDataStoreFactoryRegistry
   - Your registry manages all your custom data stores
   - Maps entity classes to data store instances

4. Configure in Spring:
   @Bean
   public FileSystemDataStoreFactoryRegistry myRegistry() {
       FileSystemDataStoreFactoryRegistry registry =
           new FileSystemDataStoreFactoryRegistry(storagePath);
       registry.registerEntityType(MyEntity.class);
       return registry;
   }

5. Done! The framework handles everything else.
